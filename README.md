## Environment

- **OS:** Windows (native; no WSL2 — see lessons-log 2026-04-26)
- **IDE:** Cursor
- **Python:** 3.12+ (3.13 in use)
- **Package manager:** [uv](https://docs.astral.sh/uv/)

## Setup

From a fresh clone:

```powershell
# 1. Install uv (one-time, per machine)
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

# 2. Sync the locked environment
uv sync

# 3. Verify
uv run python -c "import polars as pl; print(pl.__version__)"
```

Cursor will pick up the `.venv` automatically when opening notebooks; select the `quant-research (.venv)` interpreter when prompted.

## Common commands

| Task | Command |
|---|---|
| Sync env to lockfile | `uv sync` |
| Add a runtime dep | `uv add <pkg>` |
| Add a dev dep | `uv add --dev <pkg>` |
| Run tests | `uv run pytest` |
| Lint | `uv run ruff check .` |
| Format | `uv run ruff format .` |
| Launch Jupyter | `uv run jupyter lab` (or use Cursor's built-in notebook UI) |
